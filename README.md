# Easy-Schedule: Digital Diary Application

## Project Overview
**Easy-Schedule** is a fully functional, web-based digital diary application designed to streamline scheduling and time management. The platform provides users with an intuitive interface to create, track, and organize tasks, events, and schedules dynamically in real-time.

👉 **Live Demo:** [ofiromri.github.io/Easy-schedule/](https://ofiromri.github.io/Easy-schedule/)

---

## Core Features & Capabilities
* **Dynamic Calendar UI:** A clean, user-friendly digital diary layout built to optimize daily scheduling and user onboarding.
* **Real-Time Task Tracking:** Allows seamless creation and management of calendar events, ensuring an efficient workflow.
* **Front-End Architecture:** Built with a decoupled structure separating the core UI styles from the interactive routing and execution scripts.

---

## Technical Stack & Methodologies
* **Languages:** JavaScript (ES6+), HTML5, CSS3.
* **Tools & Practices:** Version control using Git and GitHub, responsive web design principles.
